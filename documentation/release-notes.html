Release notes

Every notable change to Corevex, newest first. We follow semantic versioning.

v4.0.0 Latest June 2026

Corevex — the complete Tailwind rebuild & rebrand

A ground-up rewrite of the UI layer. The template moves off Bootstrap 4 / jQuery / Laravel 9 onto a modern Laravel 12 + Tailwind CSS v4 + Alpine.js foundation, and rebrands from Radmin to Corevex.

Added
  • Brand-new Corevex identity — SaaS logo, favicon and tagline “All-in-One Laravel Admin Template for POS, Inventory & Accounting”
  • Live theme customizer with 8 modern preset palettes, per-element controls and a no-flash loader
  • 25+ reusable Tailwind + Alpine Blade components (table, drawer, modal, tabs, stat card, status pill, forms…)
  • Scalable permission selector — auto-grouping, search, per-group select-all, built for 100+ permissions
  • Drawer-based create/edit flows for Roles & Permissions
  • Topbar workspace switcher across Dashboard, Inventory, Accounting and POS
  • Product landing page, full documentation and this release-notes page (in /docs)
Changed
  • Rebuilt every screen on Tailwind CSS v4 (@theme tokens — no tailwind.config.js)
  • All interactivity migrated from jQuery to Alpine.js
  • Deep-redesigned Inventory, POS, Accounting, Reports & Settings modules
  • Modernized topbar (workspace switcher, relocated cache tools) and footer
  • Upgraded to Laravel 12 & PHP 8.2+
Removed
  • Legacy jQuery DataTables “Editable Datatable” page and feature
  • Bootstrap-only demo items (“Bootstrap Table”, “Data Table”) and their routes
  • Bootstrap 4 and jQuery dependencies from the application shell
v3.x 2023–2024

Laravel 9–11 admin starter

  • Roles & permission management, REST API (Laravel Passport)
  • Serverside DataTables with export (CSV / Excel / PDF / Copy) and editable cells
  • Inventory, POS & accounting modules on the Bootstrap 4 ThemeKit UI
v1.x – v2.x Initial releases

Original Bootstrap 4 admin template

The first CodeCanyon releases — a Laravel admin template with roles, permissions and editable datatables built on Bootstrap 4 and jQuery.
